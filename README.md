## How to use?

```bash
docker-compose up -d --build
```

Check your logs.

```bash
docker logs -f redis-stack
```

## Modules availables?

RedisJSON, RediSearch, RedisGraph, RedisTimeSeries, and RedisBloom, RedisInsight

## RedisInsight Available
